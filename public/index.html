Adir's Toys API Documentation

Toys API

HTTP Method Route Description Parameters / Request Body
GET /toys Returns a list of toys according to the query parameters. Parameters:
limit (optional): Number of toys per page (default: 5).
page (optional): Page number to return (default: 1).
sort (optional): Field to sort by (default: _id).
reverse (optional): If set to "yes," sorts in descending order.
s (optional): Search query to match against toy names, categories, or descriptions.
GET /toys/single/:id Get information about a specific toy by its ID. Parameters:
id: The ID of the toy to retrieve.
GET /toys/count get the count of the pages and items Parameters:
N/A
GET /toys/prices Filter toys by price range with optional pagination. Parameters:
limit (optional):Number of toys per page (default: 5).
page (optional):Page number to return (default: 1).
min (optional): Minimum price to filter by (default: 0).
max (optional): Maximum price to filter by (default: Infinity).
POST /toys Upload a new toy. Requires authentication. Request Body:
Toy data in the specified format:
{
    name: String,
    category: String,
    info: String,
    price: Number
}
PUT /toys/:id Update information about a specific toy by its ID. Requires authentication. Parameters:
id: The ID of the toy to update.
Request Body:
Toy data in the specified format:
{
    name: String,
    category: String,
    info: String,
    price: Number
}
DELETE /toys/:id Delete a specific toy by its ID. Requires authentication. Parameters:
id: The ID of the toy to delete.

Users API

HTTP Method Route Description Request Body
GET /users Test endpoint to check if the server is running. N/A
GET /users/myInfo Get the information for the authenticated user (requires authentication). N/A
POST /users/signup Create a new user. Request Body:
User data to be added in the specified format:
{
    name: String,
    email: String,
    password: String
}
POST /users/login Authenticate a user and receive a JWT token in response. Request Body:
User credentials in the specified format:
{
    email: String,
    password: String
}
